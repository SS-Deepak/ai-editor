# 🎨 AI Website Editor

A modern, drag-and-drop website builder built with Next.js and Tailwind CSS.
**No database required** - uses JSON files for storage (budget-friendly startup approach).

> **🤖 For AI Agents**: Read `AI_CONTEXT.md` or `QUICK_START.md` for project briefing.

## ✨ Features

- 🖱️ **Drag & Drop** - Build pages visually
- 🎨 **Color Palettes** - Auto-apply matching colors to elements
- ⚙️ **Global Settings** - Set fonts, colors, spacing once, apply everywhere
- 📦 **Element Library** - Admin-managed + user custom elements
- 📐 **Layouts** - Pre-built components & page templates
- 📱 **Responsive** - Desktop, tablet, mobile views
- 💾 **File Storage** - JSON-based, no database needed
- 🌊 **Streaming Load** - Progressive loading prevents UI freeze
- 🌙 **Dark Mode** - Material Design light/dark themes

---

## 🚀 Quick Start

```bash
cd ai-editor
npm install
npm run dev
```

Open http://localhost:3000
